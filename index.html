Guía de Instalación

Descarga todo en un solo paquete

Puedes descargar todos los archivos necesarios desde nuestro repositorio en MEGA:

Repositorio completo MushiServer

1. Instalar Hamachi

Para conexión directa al servidor:

  • Instala Hamachi desde el link de MEGA
  • Crea cuenta o inicia sesión
  • Únete a una de nuestras redes


2. Instalar Java

Requerido para Minecraft 1.21.5:

  • Descarga Java desde el link de MEGA
  • Verifica instalación con java -version

3. Usar SK Launcher

Alternativa al launcher oficial:

  • Descarga desde el link de MEGA
  • Ejecuta en modo "sin conexión"
  • Instala la version 1.21.5


4. Conectarse al Servidor

  • Hamachi: Usa la IP que aparece en Hamachi
  • Versión: Minecraft 1.21.5

Reglas del Servidor

CONVENIO URBANO DE MUSHI SERVER

Vigente desde 12 Abril 2025 hasta la finalización del SV.

En vísperas de iniciar una nueva aventura veraniega en el server de Don Pepe Giacoia, se va a realizar este presente convenio para designar puntos en común, y regular la actividad urbano-arquitectónica del server.

Requerimientos para unirse

  • Aceptar jugar al SV en comunidad, entendiendo que forma parte de un todo donde sus acciones alteran al resto
  • Mantener conductas decentes en espacios comunes y privados
  • Respetar el Código de Urbanidad
  • Prohibido actos de terrorismo, genocidio animal/humano o troleos destructivos
  • Uso de hacks/cheats prohibido - sancionado por mayoría simple (2/3)

Código Urbano

  • No construcciones voladoras descontextualizadas
  • Prohibido alterar el Skyline negativamente
  • No construir en parcelas ajenas sin consentimiento
  • Prohibidos cubos simples sin ventanas estilo soviético
  • Permitido cualquier estilo arquitectónico respetando belleza básica
  • Prohibido modificar zonas comunes sin autorización

Autoridad de Urbanidad y Estética

Conformada por 3 personas elegidas por la comunidad. Decisiones por mayoría simple (2/3).

Competencias:
  • Construcción/modificación de calles, caminos y conexiones
  • Modificaciones al Código de Urbanidad
  • Resolución de conflictos entre jugadores
  • Decisiones sobre estética general del SV
  • Modificación/supresión de construcciones que incumplan normas

Definición de Espacios

Comunes:
  • Caminos, pasillos y accesos a la ciudad
  • Zona aérea sobre estos elementos
  • Costas, bahías y zonas marítimas (200 bloques)
  • Mazmorras y estructuras generadas naturalmente
Privados:
  • Zonas propiedad de cada jugador (hogar/residencia)
  • Túneles y canales privados bajo sus casas
